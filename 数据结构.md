# 第一章 绪论
## 图灵机
图灵机组成要件：  
tape 依次均匀地划分为单元格，各注有某一字符，默认为#  
alphabet 有限的字母表  
head 读写头  
state 有限种状态  


# 单链表
## 添加新节点
new->next = cur->next;  
cur->next = new;  

## 删除节点
prev->next = cur->next;  
cur->next = NULL;  

# 双链表
## 添加节点
new->next = cur->next;  
cur->next->prev = new;  
cur->next = new;  
new->prev = cur;  

## 删除节点
cur->prev->next = cur->next;  
cur->next->prev = cur->prev;   
cur->prev = NULL;
cur->next = NULL;

# 栈
## 数组实现
stack->top++;

# 二叉树
## 性质
二叉树的第i层上至多有2^(i-1)个节点  
深度为k的二叉树至多有2^(k)-1个节点  
对于一颗非空二叉树，如果叶子节点数为n0，度为2的节点数为n2，则有n0=n2+1;  
具有n个节点的完全二叉树深度为(log2n)+1
